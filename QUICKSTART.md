# 🚀 CareerBoost 빠른 배포 가이드

## ⚡ 5분 안에 배포하기

### 1단계: GitHub에 업로드 (2분)

```bash
# 프로젝트 폴더에서 실행
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/careerboost.git
git push -u origin main
```

**YOUR_USERNAME**을 본인 GitHub 아이디로 변경하세요!

---

### 2단계: Vercel에 배포 (3분)

1. https://vercel.com 접속
2. GitHub로 로그인
3. "New Project" 클릭
4. "careerboost" 저장소 선택
5. "Deploy" 클릭
6. 완료! 🎉

---

## 📦 Claude 프로젝트로 저장

1. Claude 채팅 상단 오른쪽 **"⋮"** 클릭
2. **"Add to Project"** 선택
3. **"Create new project"** 선택
4. 이름: `CareerBoost`
5. 완료!
